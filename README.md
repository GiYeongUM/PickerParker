## 소개
<img src = "https://github.com/GiYeongUM/PickerParker/blob/main/imgs/news_img.png" width = "600px"/><br/>
- 운전을 하다보면 주차자리 찾기는 많이 번거롭습니다. 특히, 주말이나 좁은 골목길에는 대중교통이 편할때도 있습니다.
- 이 프로젝트는 주차장 등록/검색/예약이 가능하며 개인 수익창출 또한 가능한 <b>PickerParker</b>입니다.
- 개인적인 이득 뿐만 아니라 라즈베리파이를 이용해 예약된 사용자가 아니면 경고음이 출력되어 불법주차의 예방도 가능합니다.
- 이 시스템은 디바이스 - 앱 - 서버 가 서로 연동되며 검색/예약/관리/회원은 서버 - 앱끼리 HTTP 통신, 주차시에는 앱 - 디바이스가 BLE시스템으로 통신합니다.
</br>
